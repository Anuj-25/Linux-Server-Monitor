# Linux Server Monitoring and Automation System

## Objective
Automate Linux server health monitoring using Bash, Python, and Cron.

## Features
- CPU Usage Monitoring
- Memory Usage Monitoring
- Disk Usage Monitoring
- Running Service Monitoring
- System Log Collection
- Python Log Analysis
- Automated Execution using Cron

## Technologies Used
- Linux
- Bash
- Python
- Cron
- Git
